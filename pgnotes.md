# PostgreSQL Notes

## Command Line
| Command     | Description     |
|-------------|-----------------|
|psql -U >username<| connect to CLI|
|heroku pg:psql| connect to heroku db|

## Postgres CLI
 | Command     | Description     |
 |-------------|-----------------|
 |\c >database<| switch databases|
 |\conninfo    | see connection info|
 |\i >filename<| run file|
 |\q           | quit|
 |\h           | help|
